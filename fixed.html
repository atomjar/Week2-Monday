Function Explained

See the Pen pNbPpV by Adam Jahr (@atomjar) on CodePen.

This function calls for two arguments: the serviceLevel and the mealCost. Within it, the function has a variable with an array of keys that are serviceLevel strings, and numerical values associated with those keys. When run, the function returns the mealCost argument, multiplied by the tipLevel object, which is a value from the array, identified by the [serviceValue] in the brackets, plus 1. In this way, with just a string from the array's keys, and a number as the mealCost, the function can return the cost of the meal with the tip included.
